Current state, version, and Azure target for each Forms stack component
Forms 12c · WLS 12c Azure Target Mapped
Component Current State Version Azure Target Notes
Forms Runtime Oracle Forms 12c 12.2.1.x WebLogic VM or Modernized Forms 14c released Dec 2024; WLS jointly supported on Azure
App Server Oracle WebLogic Server 12c (12.2.1.4) Linux VM (BYOL) or container runtime Or integrate with Azure Front Door
Web Server Oracle HTTP Server / Apache 12c Azure App Gateway + NGINX Replace OHS with Azure App Gateway + NGINX on VM
Forms Servlet frmservlet / FormsServlet 12c Same (lift & shift VM) No change for lift & shift
PostgreSQL Data Tier Oracle DB 12.2 EE RAC 12.2.0.1 Per DB option selected Forms talks to DB via JDBC/SQLNET
Reports Oracle Reports (if any) 12c Migrate separately Reports 12c deprecated — plan migration

Session Model Stateful (Forms session) Long-lived Preserve on lift & shift container runtime profile required if containerized

SSO / Auth Oracle SSO or AD LDAP/AD Azure AD / Entra ID Configure WLS LDAP to Entra ID
License — WLS WebLogic Suite perpetual 8 proc lic assumed BYOL on Azure VM (2 vCPU = 1 lic) 8 proc lic = 16 vCPUs on Azure
License — Forms Bundled in WLS Suite Included Included in WLS BYOL Oracle Forms included in WLS Suite
Support timeline: Oracle Forms 12c Premier Support ended Dec 2024. Sustaining Support ends Dec 2028. Forms 14c (released Dec 2024) extends support to ~2034 but requires WebLogic 14c upgrade. For HRDPRD HCM scenario, lift & shift to Azure VM is the fastest path; modernization to React is the strategic long-term play.
